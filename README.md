# MohitKhare PHP Utilities

Developer utilities and token counting tools for PHP by [Mohit Khare](https://mohitkhare.me).

## Installation

```bash
composer require arnaudleroy-studio/mohitkhare
```

## Usage

```php
<?php
require 'vendor/autoload.php';

$client = new MohitKhare\Client();
echo $client->getVersion(); // "0.1.0"
echo $client->getBaseUrl(); // "https://mohitkhare.me"
```
